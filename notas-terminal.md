4- Comandos de terminal
pwd definicion: Muestra la ruta de la carpeta donde estoy parado.
pwd: Lo use para confirmar en que carpeta se encontraba.

Is definicion: Lista los archivos de la carpeta actual.
Is: Con este comando liste mis archivos para verificar que el index.html y style.css esten bien guardados.

cd Desktop definicion: Cambia el directorio hacia el Escritorio.
cd: Lo usé para moverme al Escritorio y tener mi carpeta del TP a mano.

touch definicion: Crea archivos vacíos.
touch: Use este comando para crear el archivo de mi primera sesion en la terminal. 

mkdir definicion: Crea una carpeta nueva.
mkdir: Cree la carpeta principal donde guarde todo mi proyecto.

code definicion:  Abre el Visual Studio Code en la carpeta actual.
code: Fue clave para abrir mi proyecto directamente en el editor y empezar a programar.

git config --global user.name "Mi Nombre" definicion: Configura mi nombre de usuario en Git.
git config --global user.name "Mi Nombre": Lo usé en el paso de configuración para que mis trabajos queden firmados con mi nombre.
git config --global user.email "mi@mail.com" definicion: Vincula mi correo electrónico a Git.
git config --global user.email "mi@mail.com" : Lo usé para completar mi perfil de desarrollador antes de empezar a usar Git.

git config--list definicion: Muestra toda la configuración actual de Git.
git config--list: Lo ejecuté al final para asegurarme de que mi nombre, mi mail y el editor quedaron bien guardados.
