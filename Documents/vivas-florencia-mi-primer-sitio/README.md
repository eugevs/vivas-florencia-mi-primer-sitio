Reflexión sobre la terminal
Me parece que usar la terminal es clave porque una vez que te aprendés los comandos, manejás los archivos mucho más rápido que con el mouse. Siento que tengo más control sobre lo que hago y es una herramienta que voy a usar siempre como programadora. Además, es la forma más profesional de usar Git sin que se trabe.

¿Por que es importante aprender a usar la terminal si existe la interfaz grafica? 
Es importante porque me permite trabajar con mucha más rapidez y precisión que usando solo el mouse. Me da un control total sobre mis archivos y es una herramienta fundamental en el mundo profesional, ya que muchos servidores no tienen interfaz gráfica. Además, es la mejor forma de usar Git sin errores y automatizar tareas que manualmente llevarían mucho tiempo.
